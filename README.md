### Featured Project

#### [Steam Analytics — Microsoft Fabric](https://github.com/gatotroller/steam-analytics-fabric)
An enterprise-grade data engineering project built on **Microsoft Fabric** with **medallion architecture** (Bronze → Silver → Gold).

**Highlights:**
- Async HTTP ingestion from Steam API with rate limiting
- Data validation with **Pydantic** and **SCD Type 2** change tracking
- **Repository** and **Strategy** design patterns for maintainability
- Structured logging and comprehensive test coverage
- **Delta Lake** storage with incremental processing
